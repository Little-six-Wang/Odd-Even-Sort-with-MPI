# Odd-Even-Sort-with-MPI
利用MPI实现奇偶排序算法
# 运行
在第一个目录（即只有一个makefile文件的目录）运行`make run`
# 修改参数
你可以通过修改`include`目录中的`parameters.h`中的`N`的值来改变问题的规模，可以通过修改第一个目录中的`makefile`中的`THREAD`来改变线程的数量
